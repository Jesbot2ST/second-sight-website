03 — Spacing System

8px Grid

All spacing is based on an 8px unit. Use multiples only — don't freestyle it.

sp1
8px — tight gaps, inline elements
sp2
16px — card padding (small), stacked labels
sp3
24px — between related elements
sp4
32px — card padding, section sub-gaps
sp5
48px — between sections (mobile)
sp6
64px — section padding (desktop)
sp7
96px — hero padding, major section gaps
Container widths

Max content width: 1100px
Narrow / reading width: 680px
Side padding (desktop): 48px
Side padding (mobile): 20px
Border radius: 3–4px (subtle, not rounded)
